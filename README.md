# Agentic Vue Books v3

A minimal Vue 3 + Vite app used for an AI coding agents workshop.

## Setup

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (usually http://localhost:5173).

## Exercise

Use an AI coding agent to implement the "Favorite Books" feature:

- Add an `isFavorite` boolean to each book
- Show a star icon next to each book and toggle it on click
- Persist favorite state (e.g., in localStorage or via the in-memory API)
- Add a "Show favorites only" toggle or "Sort by favorites" control
